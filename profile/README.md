# Clark College Software Engineering

This GitHub organization hosts course materials, projects, and resources for the Software Engineering courses in [Clark College's](https://www.clark.edu/) Computer Science & Engineering program.

## Courses

**CSE 310 – Software Engineering**  
Practical experience across the software development lifecycle—from requirements analysis to release. Topics include specification, design, abstraction, programming style, testing, maintenance, and software project management, with emphasis on effective teamwork.

**CSE 325 – Software Design & Development**  
Intermediate software design and development with emphasis on user interface design, software architectures, and design patterns.
